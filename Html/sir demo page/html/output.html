Output

  • The output element in HTML is used to represent the result of a calculation or the outcome of a user action.
  • It is often used in conjunction with JavaScript to display dynamically generated content or the result of calculations within a web page.
  • The output element can be updated and modified through JavaScript to display different values based on user input or program logic.

Example :

Enter two numbers:

Sum: 0

Explanation :

  1. Two input elements of type number allow the user to enter numeric values.
  2. A button element with an onclick attribute is used to trigger the calculateSum() JavaScript function when clicked.
  3. The output element with the id "result" is initially set to display "0" as its default value.
  4. The script section contains a JavaScript function calculateSum() that calculates the sum of the two input values and updates the content of the output element with the result.
  5. When you enter two numbers in the input fields and click the "Calculate Sum" button, the JavaScript function will be executed. It retrieves the input values, performs the addition, and then displays the result in the output element.
  6. This example demonstrates how the output element can be used to display dynamic content or calculation results within an HTML document.
